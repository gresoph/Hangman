## Проект 1. «Виселица» (Hangman) – классическая игра
1. Название
Hangman Console Game

2. Техническое задание

Программа загадывает случайное слово из заранее заданного списка (5–10 слов).
Игрок видит маску слова: неизвестные буквы заменены на _ (например, _ _ _ _).
Игрок вводит одну букву. Если она есть в слове – открываются соответствующие позиции. Если нет – уменьшается количество попыток (например, 6 попыток).
Игра завершается победой, когда открыты все буквы, или поражением, когда попытки закончились.
После окончания игры выводится сообщение о результате и показывается загаданное слово.
Дополнительно (по желанию): отображение уже введённых ошибочных букв, защита от повтора букв, регистронезависимость.
3. Цель проекта
Закрепить работу со строками (срезы, поиск), списками (маскировка), множествами (хранение ошибочных букв), циклами (основной игровой цикл) и условиями. Научиться выносить логику в функции (генерация маски, проверка буквы, отрисовка состояния).

4. Краткое описание
Консольная игра в «Виселицу», где игрок угадывает слово по буквам. Проект объединяет все базовые конструкции Python в простом, но увлекательном игровом процессе.
